# Student Performance SQL Analysis

This project is my first SQL analysis project built using MySQL Workbench. I worked on a student performance dataset to practice writing SQL queries and understand how data can be used to answer real-world questions.

Instead of only writing practice queries, I tried to approach the dataset like a data analyst by asking business questions and using SQL to find meaningful insights.

---

## Dataset

**Dataset:** Student Performance Factors

The dataset contains information about students, including:

- Exam Score
- Attendance
- Hours Studied
- Motivation Level
- Teacher Quality
- Internet Access
- School Type
- Parental Involvement
- Sleep Hours
- Previous Scores
- and other academic factors.

---

## What I worked on

### Basic Analysis

- Total number of students
- Average, maximum and minimum exam scores
- Average attendance
- Top and bottom performers
- Filtering records using different conditions
- Grouping and aggregation

### Business Analysis

Some of the questions answered include:

- Does motivation level affect exam scores?
- Does parental involvement improve student performance?
- Does teacher quality make a difference?
- Does internet access impact academic results?
- Which school type performs better on average?
- Which students perform above the average exam score?
- Which students may require additional academic support?

## How to Run

1. Import the dataset into MySQL Workbench using the Table Data Import Wizard.
2. Open the SQL files.
3. Execute the queries.
4. Review the outputs and insights generated from the analysis.
